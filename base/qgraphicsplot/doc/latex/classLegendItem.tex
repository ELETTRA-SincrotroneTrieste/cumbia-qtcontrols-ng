\doxysubsection{Detailed Description}
This class is created and managed by the \doxyref{Plot\+Scene\+Widget}{p.}{classPlotSceneWidget}.

\begin{DoxyParagraph}{Configuration}
The legend can be configured by changing its z\+Value (default value is 100), by filling its background and drawing a black bounding rect around (both disabled by default.
\end{DoxyParagraph}
\begin{DoxyParagraph}{Accessing the Legend\+Item.}
Use \doxyref{Plot\+Scene\+Widget\+::legend\+Item()}{p.}{classPlotSceneWidget_afb68ccc9b20bc177d2e10b33a99d89ec}
\end{DoxyParagraph}
\begin{DoxyParagraph}{Curve aliases.}
The legend is able to show an alias instead of the complete source name of the curve. Use set\+Curve\+Alias to setup an alias for each curve. set\+Curve\+Alias can be invoked before or after TPlot\+Scene\+Widget\+::set\+Sources\+List 
\end{DoxyParagraph}
