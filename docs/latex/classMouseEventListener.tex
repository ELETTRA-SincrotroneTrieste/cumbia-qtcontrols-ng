\doxysubsection{Detailed Description}
an abstract class that can be used to be notified when a mouse event takes place 

Mouse press, move, release and click event can be intercepted. In each method, a reference to the \doxyref{Plot\+Scene\+Widget}{p.}{classPlotSceneWidget} that received the event is passed.
