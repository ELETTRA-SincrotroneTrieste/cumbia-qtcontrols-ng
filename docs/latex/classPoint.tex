\doxysubsection{Detailed Description}
a point item that calculates its position in the view and simply draws a cross centered at x,y and the represented x and y values. 

You must inherit from this class or install a custom \doxyref{Item\+Painter\+Interface}{p.}{classItemPainterInterface} to change its drawing behaviour.
