\doxysubsection{Detailed Description}
A QGraphics\+Object that can be moved and can be attached to a writer in order to write a value according to the position of the item. 

A \doxyref{Target\+Item}{p.}{classTargetItem} must be attached to a couple of \doxyref{Scale\+Item}{p.}{classScaleItem} axes, and must be constructed passing a reference to a \doxyref{Plot\+Scene\+Widget}{p.}{classPlotSceneWidget}, as you can see in the \doxyref{Target\+Item}{p.}{classTargetItem} constructor.

\begin{DoxySeeAlso}{See also}
\doxyref{Target\+Item}{p.}{classTargetItem}
\end{DoxySeeAlso}
In order to receive move events, you have to implement the \doxyref{Item\+Move\+Listener}{p.}{classItemMoveListener} interface and install it by calling install\+Item\+Move\+Listener. Move events are sent only if the mouse button is pressed. set\+Pos does not per se produce a move event.

\begin{DoxySeeAlso}{See also}
\doxyref{install\+Item\+Move\+Listener}{p.}{classTargetItem_ad5ab5abecc38a3a7444db71ffb8f1bcf} 

\doxyref{Item\+Move\+Listener}{p.}{classItemMoveListener}
\end{DoxySeeAlso}
\begin{DoxyParagraph}{Automatic repositioning of the Target\+Item when the plot geometry changes.}
\doxyref{Target\+Item}{p.}{classTargetItem} implements \doxyref{Axis\+Change\+Listener}{p.}{classAxisChangeListener} interface. When the \doxyref{Plot\+Scene\+Widget}{p.}{classPlotSceneWidget} plot canvas changes its geometry, the canvas\+Rect\+Changed method repositions the target item according to the new geometry and the values of x and y axes coordinates of the target items, which are set with setX and setY.
\end{DoxyParagraph}
\begin{DoxyParagraph}{Note}
The \doxyref{Target\+Item}{p.}{classTargetItem} constructor {\bfseries{installs the \doxyref{Target\+Item}{p.}{classTargetItem} on each \doxyref{Scale\+Item}{p.}{classScaleItem} in order to receive the geometry change events}} of the \doxyref{Plot\+Scene\+Widget}{p.}{classPlotSceneWidget} and to be able to reposition itself when the canvas rect changes. 
\end{DoxyParagraph}

Initiates a red \doxyref{Target\+Item}{p.}{classTargetItem} with an inner radius of 15 and an outer radius of 20. The item is movable, selectable, sends geometry changes and sends scene position changes (QGraphics\+Item\textquotesingle{}s item flags).

The position with respect to the x and y scales is set to 0.\+0 by default. Remember to call setX and setY when you know what the position of the item is. Storing x and y position in axes coordinates is important in order to reposition the item when the canvas rect of the plot changes.

\begin{DoxyParagraph}{Automatic repositioning of the Target\+Item on plot geometry change event.}
When the \doxyref{Plot\+Scene\+Widget}{p.}{classPlotSceneWidget} plot canvas changes its geometry, the canvas\+Rect\+Changed method repositions the target item according to the new geometry and the values of x and y axes coordinates of the target items, which are set with setX and setY The \doxyref{Target\+Item}{p.}{classTargetItem} constructor installs the \doxyref{Target\+Item}{p.}{classTargetItem} on each axis, x\+Axis and y\+Axis, in order to automatically manage the object repositioning when the canvas plot rect changes. 
\end{DoxyParagraph}
