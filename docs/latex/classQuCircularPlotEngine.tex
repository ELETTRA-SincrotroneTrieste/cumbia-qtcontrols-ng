\doxysubsection{Detailed Description}
Engine shared by either Qu\+Circular\+Plot\+Item or Qu\+Circular\+Plot\+Widget that stores configuration and does all the painting and zooming and stores the data. 

Qu\+Circular\+Plot\+Item\+::engine and Qu\+Circular\+Plot\+Widget\+::engine return a reference to their \doxyref{Qu\+Circular\+Plot\+Engine}{p.}{classQuCircularPlotEngine}, that shall be used by clients to access all operations, as well as signals and slots.

\begin{DoxyParagraph}{Plot configuration}
Plot settings are held by the {\itshape \doxyref{qucircularplot\+\_\+attributes}{p.}{structqucircularplot__attributes}} public member named {\itshape a} \doxyref{Qu\+Circular\+Plot\+Engine\+::att}{p.}{classQuCircularPlotEngine_ae555d89cb571bc69bfbddff1569b13d9} and \doxyref{Qu\+Circular\+Plot\+Engine\+::attributes}{p.}{classQuCircularPlotEngine_a8e67a8b3199441068520421eaba3db41} are convenience methods that return a (non const) reference to {\itshape a}.

The \doxyref{Qu\+Circular\+Plot\+Drawable\+\_\+\+I\+::z}{p.}{classQuCircularPlotDrawable__I_a5eccaad178d2e49cbc930972b10c4ad9} value determines the order in which each drawable is painted. Plot background is drawn before everything else. Then drawables with negative z are drawn. Axes, bounds, curves follow. Drawables with positive z are then painted.

Drawables that {\itshape scale} are drawn according to the painter transformation (zoom, if applied). Those that do not scale are drawn before the transformation is applied to the painter or after the painter is restored (always according to the {\itshape z position}.

For example, the y value of the selected point can be shown at the center of the plot (\doxyref{Qu\+Circular\+Plot\+Selection\+Value}{p.}{classQuCircularPlotSelectionValue}), or the plot can be divided in \char`\"{}sectors\char`\"{} (\doxyref{Qu\+Circular\+Plot\+Disk\+Sector}{p.}{classQuCircularPlotDiskSector}).
