\doxysubsection{Detailed Description}
This is an interface that you can subclass to be notified when the plot rect changes. 

Subclass Plot\+Event\+Listener and implement the plot\+Rect\+Changed method if you want to be notified when the plot rect changes.

This usually happens when the scene rect changes or the plot area changes.
